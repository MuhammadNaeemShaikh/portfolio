## MouseTrackingCV

> Developed an AI-based system to control the mouse cursor using Python and OpenCV with the real-time camera. Fingertip location is mapped to RGB images to control the mouse cursor.

## Dependencies  -
* Python
* openCV
* mediapipe
* autopy
* numpy
* pandas
